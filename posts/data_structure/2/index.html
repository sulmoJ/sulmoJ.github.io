Heap

완전 이진 트리의 일종으로 최대값이나 최솟값을 찾는데 최적화된 자료구조

특징

  • 완전 이진트리의 일종이며, 우선순위 큐를 위해 만들어짐
  • 반정렬 상태를 유지한다.(부모노드보다 자식노드가 작다/크다 로 유지)
  • 중복된 값을 허용한다.(이진 탐색 트리는 허용 안함)

힙의 종류

  • 최대 힙(max heap): key(부모 노드) >= key(자식 노드)을 만족하는 완전 이진 트리
  • 최소 힙(min heap): key(부모 노드) <= key(자식 노드)을 만족하는 완전 이진 트리

삽입과 삭제

* 최대힙을 기준으로 구현 및 설명 했습니다.

힙을 구현시 기본적으로 배열에 저장을 하며, 삽입과 삭제 과정이 있다. 완전 이진트리이기 때문에 부모와 자식의 접근에 있어서 점화식이 정해져 있고 이를 활용하여 비교하며 위치를 잡는다.

  • 왼쪽 자식의 인덱스 = (부모의 인덱스) * 2
  • 오른쪽 자식의 인덱스 = (부모의 인덱스) * 2 + 1
  • 부모의 인덱스 = (자식의 인덱스) / 2
삽입
  • 배열의 가장 끝에 데이터를 추가한다.
  • 부모와 비교 후 자신보다 작다면 swap한다.
  • 부모노드의 값이 더 클때까지 반복
삭제
  • 최대 값인 루트를 삭제한다.
  • 배열의 가장 끝에 데이터를 트리의 루트로 가져온다.
  • 자식 노드와 비교해 자신보다 큰 값과 swap한다.(자식 노드중에서도 더큰 자식 노드와 swap한다)
import sys

// 삽입
def insert_heap(_heap, num):
    _heap.append(num)
    current_key = len(_heap)-1
    while(current_key > 1):
        if(current_key % 2):
            if _heap[(current_key-1)//2] < _heap[current_key]:
                _heap[(current_key-1)//2], _heap[current_key] = _heap[current_key], _heap[(current_key-1)//2]
                current_key = (current_key-1)//2
            else:
                break
        else:
            if _heap[current_key//2] < _heap[current_key]:
                _heap[current_key//2], _heap[current_key] = _heap[current_key], _heap[current_key//2]
                current_key = current_key//2
            else:
                break

// 삭제
def delete_heap(_heap):
    max_val = _heap[1]
    tmp = _heap.pop()

    parant = 1
    child = 2

    while(child <= len(_heap)-1):
        if child < len(_heap)-1 and _heap[child] < _heap[child+1]:
            child += 1

        if tmp >= _heap[child]:
            break

        _heap[parant] = _heap[child]

        parant = child
        child *= 2

    if len(_heap) != 1:
        _heap[parant] = tmp
    return max_val


n = int(sys.stdin.readline())

_heap = [0]

for _ in range(n):
    num = int(sys.stdin.readline())
    if num == 0:
        if len(_heap) == 1:
            print(0)
        else:
            print(delete_heap(_heap))

    else:
        insert_heap(_heap, num)
